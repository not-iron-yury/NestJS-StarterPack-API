# NestJS Auth API Starter

Готовый к использованию стартовый проект аутентификации, созданный с использованием NestJS, Prisma и PostgreSQL.

## Features

NestJS + TypeScript
Prisma ORM
PostgreSQL
JWT Access & Refresh Tokens
Cookie-based authentication
Email verification
Password reset
Zod validation
Repository pattern
Centralized configuration
Environment validation
Production-ready project structure

Prisma ORM
